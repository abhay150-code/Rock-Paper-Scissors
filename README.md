# 🪨 Rock Paper Scissors - Modern React Edition

A sleek, responsive, and interactive Rock-Paper-Scissors game built with React and Vite. Experience the classic game with a modern UI and real-time statistics.

## ✨ Features

- **Modern UI**: Clean design with vibrant gradients and glassmorphism-inspired cards.
- **Score Tracking**: Keep track of your wins vs. the computer in real-time.
- **Win Streaks**: See how many consecutive games you can win.
- **Game History**: Scroll through your previous moves and results.
- **Reset Functionality**: Easy reset to start a fresh session.
- **Responsive Design**: Works perfectly on different screen sizes.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abhay150-code/Rock-Paper-Scissors.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Rock-Paper-Scissors
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
